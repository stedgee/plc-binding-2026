# plc-binding-2026
Startup and Configuration files for PLC Geometrical Route (peripheral membrane-binding proteins in homogenous lipid compositions)


inputs directory contains NAMD config, Colvars config, necessary parameter and topology, psf and pdb, and other files (.coor, .vel, .xsc) necessary to run separation and tilt simulations.

Separation inputs are broken down by window of PMF (15 Angstrom each).
